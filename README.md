# Harry The Fairy Pc Version
I ported harry the fairy to pc, and added some more features

# Download:
Download the zip file [here](https://github.com/thebe-stone/Harry-The-Fairy-Pc-Version/releases/download/game/Harry.1.1.zip) and run the exe file inside to play

# New Features:
- Keyboard controls using WASD and Arrow keys
- new Demon Mode (removes checkpoints and shields)
- You dont need to use an android emulator
- All in app purchases are now free (Zaxis please don't sue me)
